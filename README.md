# All Things Current

This is Version 3 of the "All Things Current" website.

## How to Deploy

### Using GitHub Pages
1. Create a repository on GitHub.
2. Upload all files in this folder (`index.html` and `style.css`).
3. Go to **Settings → Pages → Source → main branch → / (root)**.
4. Save. Your website will be live at:
https://nicknineLA.github.io/all-things-current/
